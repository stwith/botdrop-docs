## Guided Setup

A 4-step wizard walks you through everything: authentication, agent configuration, installation, and channel connection. No command line knowledge required.

## Multi-Provider Support

BotDrop works with all major AI providers:

| Provider | Models |
|----------|--------|
| Anthropic | Claude Opus, Sonnet, Haiku |
| OpenAI | GPT-4o, GPT-4, GPT-3.5 |
| Google | Gemini Pro, Gemini Flash |
| OpenRouter | 100+ models |
| Custom | Any OpenAI-compatible API |

## Messenger Integration

Connect your agent to:
- **Telegram** — Personal or group chats
- **Discord** — Server channels and DMs

Your agent responds in real time, just like chatting with a friend.

## Background Gateway

BotDrop runs a persistent gateway service that:
- Keeps your agent alive even when the app is in the background
- Auto-restarts on crash or reboot
- Manages resource usage to preserve battery

## Built on OpenClaw

Under the hood, BotDrop uses [OpenClaw](https://github.com/nicepkg/openclaw), a powerful AI agent framework. This means your agent has access to tools, memory, skills, and everything else OpenClaw supports.
