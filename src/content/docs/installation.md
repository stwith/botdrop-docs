## Download

Get the latest APK from [GitHub Releases](https://github.com/zhixianio/botdrop-android/releases).

## Requirements

- Android 10 or later
- At least 2 GB free storage (OpenClaw + Node.js takes ~500 MB)

## Install

1. Download the APK file
2. Open it on your Android device
3. If prompted, allow installation from unknown sources
4. Tap **Install**

Once installed, open BotDrop and proceed to [Setup →](/setup/).
