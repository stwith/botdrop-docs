## 创建 Telegram Bot

1. 打开 Telegram，搜索 [@BotFather](https://t.me/BotFather)
2. 发送 `/newbot`
3. 按提示设置名称和用户名
4. BotFather 会给你一个 **Bot Token**（类似 `123456:ABC-DEF1234ghIkl-zyx57W2v1u123ew11`）
5. 复制这个 Token

## 在 BotDrop 中配置

1. 在频道步骤中，选择 **Telegram**
2. 粘贴 Bot Token
3. 点击 **连接**

Bot 现在在 Telegram 上线了。发条消息试试。

## 群聊

在群组中使用 Bot：

1. 把 Bot 添加到群组
2. 用 `@你的bot用户名` 提及它，或把它设为群管理员
3. Bot 会回复提及它的消息

## 小贴士

- 在 BotFather 中用 `/setuserpic` 和 `/setdescription` 自定义 Bot 的头像和描述
- 要让 Bot 读取群里的所有消息（不只是 @），在 BotFather 中用 `/setprivacy` 关闭隐私模式
