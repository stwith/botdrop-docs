## BotDrop 是什么？

BotDrop 是一款免费的 Android 应用，让你在手机上一键安装和运行 OpenClaw 等 AI 机器人。提供可视化界面，无需编程即可配置。

## 需要 root 吗？

不需要。BotDrop 作为普通 Android 应用运行，不需要 root 权限。

## 需要自己的服务器吗？

不需要！BotDrop 让你直接在手机上配置 API 密钥和连接聊天软件，省去了租服务器、配环境的麻烦。

## 支持哪些 AI 模型？

支持 100+ 模型，包括 Claude、GPT-4、Gemini、DeepSeek、Llama 等。可以使用各种 AI 提供商，也可以配置自定义 API 端点。

## 可以连接哪些聊天平台？

目前支持：
- Telegram
- Discord
- 飞书

更多平台正在开发中。

## 流量消耗大吗？

很小。文字对话每条消息约 1-5KB。主要成本是 AI API 调用费用，不是手机流量。

## 手机锁屏后还能用吗？

可以。BotDrop 以后台服务运行，屏幕关闭后继续响应消息。

## 支持 iPhone 吗？

目前仅支持 Android。

## 如何在手机上安装 OpenClaw？

从 [botdrop.app](https://botdrop.app) 下载 BotDrop APK 并安装，然后通过可视化界面配置 OpenClaw，整个过程只需几分钟。

## 免费吗？

是的，BotDrop 完全免费。你只需支付 AI API 使用费。

## BotDrop 和在服务器上运行有什么区别？

| | BotDrop | 服务器 |
|---|---|---|
| 配置时间 | 5 分钟 | 30+ 分钟 |
| 技术要求 | 无 | Linux、终端 |
| 每月成本 | ¥0 | ¥30-150 VPS |
| 配置方式 | 可视化界面 | 编辑 YAML 文件 |
| 便携性 | 随身携带 | 固定位置 |

## 可以用来做手机自动化吗？

可以！BotDrop 有自动化功能，让你的 AI 控制手机，包括点击、滑动、与 App 交互。详见[自动化指南](/zh/automation/)。
