## BotDrop 是什么？

BotDrop 是一款免费的 Android 应用，让你在手机上直接运行 AI 机器人。不需要服务器，不需要编程。基于 OpenClaw 开发，提供可视化界面配置一切。

## 需要 root 吗？

不需要。BotDrop 作为普通 Android 应用运行，不需要 root 权限。

## 需要自己的服务器吗？

不需要！这是 BotDrop 的核心优势。AI 智能体完全运行在你的手机上，你只需要一个 AI 提供商的 API 密钥（比如 OpenRouter）。

## 支持哪些 AI 模型？

支持 100+ 模型，包括 Claude、GPT-4、Gemini、DeepSeek、Llama 等。你可以用 OpenRouter 一个 API 密钥访问所有模型，也可以配置自定义 API 端点。

## 可以连接哪些聊天平台？

目前支持：
- Telegram
- Discord
- 飞书

更多平台正在开发中。

## 流量消耗大吗？

很小。文字对话每条消息约 1-5KB。主要成本是 AI API 调用费用，不是手机流量。

## 手机锁屏后还能用吗？

可以。BotDrop 以后台服务运行，屏幕关闭后继续响应消息。

## 支持 iPhone 吗？

暂不支持。BotDrop 目前仅支持 Android。iOS 版本在规划中。

## 如何在手机上安装 OpenClaw？

只需从 [botdrop.app](https://botdrop.app) 下载 BotDrop APK 并安装。BotDrop 通过可视化界面自动处理所有 OpenClaw 配置。

## 免费吗？

是的，BotDrop 完全免费。你只需支付 AI API 使用费（给 OpenRouter、OpenAI 或 Anthropic 等提供商）。

## BotDrop 和在服务器上运行 OpenClaw 有什么区别？

| | BotDrop | 服务器 |
|---|---|---|
| 配置时间 | 5 分钟 | 30+ 分钟 |
| 技术要求 | 无 | Linux、终端 |
| 每月成本 | ¥0 | ¥30-150 VPS |
| 配置方式 | 可视化界面 | 编辑 YAML 文件 |
| 便携性 | 随身携带 | 固定位置 |

## 可以用来做手机自动化吗？

可以！BotDrop 有自动化功能，让你的 AI 控制手机，包括点击、滑动、与 App 交互。详见[自动化指南](/zh/automation/)。
