## 引导式设置

四步向导带你完成所有配置：认证、Agent 配置、安装、频道连接。不需要任何命令行知识。

## 多供应商支持

BotDrop 支持所有主流 AI 供应商：

| 供应商 | 模型 |
|--------|------|
| Anthropic | Claude Opus, Sonnet, Haiku |
| OpenAI | GPT-4o, GPT-4, GPT-3.5 |
| Google | Gemini Pro, Gemini Flash |
| OpenRouter | 100+ 模型 |
| 自定义 | 任何 OpenAI 兼容 API |

## 通讯工具集成

连接你的 Agent 到：
- **Telegram** — 私聊或群聊
- **Discord** — 服务器频道和私信

Agent 实时响应，就像和朋友聊天一样。

## 后台网关

BotDrop 运行持久化网关服务：
- 即使 App 在后台也保持 Agent 运行
- 崩溃或重启后自动恢复
- 管理资源使用，节省电量

## 基于 OpenClaw

底层使用 [OpenClaw](https://github.com/nicepkg/openclaw)，一个强大的 AI Agent 框架。这意味着你的 Agent 可以使用工具、记忆、技能以及 OpenClaw 支持的一切能力。
