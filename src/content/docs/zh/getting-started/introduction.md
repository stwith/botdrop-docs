BotDrop 是一个 AI Agent 托管与管理平台。

## 什么是 BotDrop？

BotDrop 让部署、管理和监控 AI Agent 变得简单。无论你运行单个 bot 还是协调一整个 agent 编队，BotDrop 提供基础设施，让你专注于 agent 本身的能力。

## 核心特性

- **Agent 托管** — 零运维部署和运行 AI Agent。
- **多 Agent 管理** — 在一个面板中监控和管理所有 agent。
- **可扩展** — 接入你自己的模型、工具和工作流。
