## 工作原理

BotDrop 把 [OpenClaw](https://openclaw.ai/) 打包成 Android App。OpenClaw 是一个 AI Agent 框架，给你的 Bot 提供工具、记忆和技能。BotDrop 处理所有安装配置，你不用碰终端。

## 支持的 Agent

| Agent | 说明 | 状态 |
|-------|------|------|
| [OpenClaw](https://openclaw.ai/) | 强大的 AI Agent 平台，支持执行能力和长期记忆 | ✅ 已支持 |
| [OwliaBot](https://owlia.bot/) | 更轻量、更安全的加密原生 OpenClaw 🦞 | 🔜 即将支持 |
| [NanoClaw](https://github.com/gavrielc/nanoclaw) | 极简容器化 Claude 助手 🦐 | 🔜 即将支持 |
| [nanobot](https://github.com/HKUDS/nanobot) | 轻量级本地 AI Bot | 🔜 即将支持 |
